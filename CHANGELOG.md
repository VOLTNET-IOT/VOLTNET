# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-15

### Added
- Initial release of VOLTNET SDK
- `VoltnetClient` - Main client for VOLTNET API
- `EnergyMeter` - Energy measurement and device management
- `PricingEngine` - Dynamic pricing and cost calculations
- `P2PMarket` - Peer-to-peer energy trading marketplace
- `TransactionManager` - Transaction and settlement management
- Comprehensive TypeScript type definitions
- Real-time WebSocket support for live updates
- Utility functions for energy calculations
- Full documentation and examples
- Three example implementations:
  - Basic usage
  - Solar energy trading
  - EV charging optimization

### Features
- Submit and query energy measurements
- Register and manage IoT devices
- Get real-time and historical pricing
- Create and accept P2P market offers
- Track transactions and settlements
- Real-time event subscriptions
- Automatic retry logic with exponential backoff
- Balance and statistics tracking
- Location-based offer searching
- CO2 emissions calculations

### Documentation
- Comprehensive README with examples
- API reference documentation
- Contributing guidelines
- MIT License
